The following list shows the platforms supported in this release.  If you're working with a version older than these, consult the [documentation archive](https://docs.tigera.io/archive) or contact Support.

### Supported {variables.prodname} features

If your platform is listed below, the features in this release will work for your platform unless an individual feature topic explicitly calls out a vendor or {variables.prodname} limitation.
